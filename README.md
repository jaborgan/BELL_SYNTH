# BELL_SYNTH

### this is the latest build of the akebono synth with integrated visualizer and ADSR envelope for modification

features a pre-structured pentatonic scale based on the akebono traditional japanese tuning
the fundamental frequency is two sine waves

layered on top are between 3 -8 harmonics that are picked randomly from SAW, Square, Sine oscillators that give the app timbral variation

future deployments might have midi integration
